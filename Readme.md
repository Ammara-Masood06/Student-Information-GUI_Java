# Student Information System - Java Swing GUI

A lightweight Desktop application developed in Java using the Swing library. This project focuses on creating a structured User Interface (GUI) for data entry, specifically designed for university student enrollment.

## Features
* **Custom Layout:** Uses absolute positioning (`null` layout) for precise control over component placement.
* **Component Integration:** Includes `JTextField` for text input, `JLabel` for headers, and `JRadioButton` for gender selection.
* **Visual Styling:** Customized background colors, custom `Garamond` fonts, and stylized buttons.
* **Non-Resizable Window:** The interface is locked to specific dimensions to maintain UI integrity.

## Technologies Used
* **Language:** Java
* **Framework:** Swing (javax.swing)
* **API:** Abstract Window Toolkit (java.awt)

## How to Run
1. Ensure you have the Java Development Kit (JDK) installed.
2. Compile the project: `javac Project1.java`
3. Run the application: `java project1.Project1`
